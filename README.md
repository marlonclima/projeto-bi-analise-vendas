# projeto-bi-analise-vendas
Projeto: Análise de Vendas

Este projeto tem como objetivo analisar dados de vendas
utilizando Power BI.

Ferramentas utilizadas:
- Excel
- Power BI

Análises realizadas:
- Vendas por categoria
- Vendas por região
- Top produtos
- Evolução das vendas
## AC2 - Evolução do Projeto

Nesta etapa foram adicionadas novas análises:

- Lucro por categoria
- Quantidade vendida por produto
- Vendas por cidade
- Melhorias visuais no dashboard

O objetivo foi aprofundar a análise de dados e gerar novos insights.
